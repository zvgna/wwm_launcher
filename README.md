# WWMRU — лаунчер русификатора Where Winds Meet

**WWMRU** — это простой графический лаунчер для установки и обновления русификации игры **Where Winds Meet**.  
Программа автоматически загружает переводы из GitHub Releases и подменяет файлы локализации в папке игры.

Лаунчер не модифицирует исполняемые файлы игры и работает только с файлами перевода.

---

## Возможности

- установка последней версии русификации из GitHub
- автоматическая проверка обновлений при запуске
- откат на одну из **последних 5 версий перевода**
- сохранение **оригинальных файлов игры** (бэкап) перед первой установкой
- запоминание пути к игре
- работа без Steam / сторонних лаунчеров
- сворачивание в системный трей
- аккуратный интерфейс без системной рамки

---

## Как работает

WWMRU загружает файлы перевода из раздела **GitHub Releases** и заменяет их в папке игры:
Where Winds Meet\Package\HD\oversea\locale\

Используемые файлы:
- `translate_words_map_en`
- `translate_words_map_en_diff` (если присутствует в релизе)

---

## Установка и использование

1. Скачайте файл `WWMRU.exe` из раздела **Releases**
2. Запустите программу
3. На вкладке **Установка** укажите папку с игрой  
   (можно выбрать как корень игры, так и папку уровнем выше)
4. Нажмите **«Установить русификатор»**
5. Дождитесь завершения установки

После установки в интерфейсе будет отображаться текущая версия перевода.

---

## Откат версии

На вкладке **Настройки** доступен выбор версии перевода:

- лаунчер автоматически подгружает **последние 5 релизов**
- можно установить любую из них
- текущая установленная версия помечается в списке

Это позволяет безопасно вернуться на предыдущую версию перевода.

---

## Резервное копирование

Перед **первой установкой** лаунчер может сохранить оригинальные файлы игры.

Бэкап создаётся один раз и хранится в папке:
WWMRU/backup/original/


Опцию можно отключить в настройках.

---

## Поведение окна

- **—** — свернуть в системный трей
- **×** — полностью закрыть программу
- клик по иконке в трее — восстановить окно
- выход из трея — завершение приложения

---

## Требования

- Windows 10 / 11
- установленная игра *Where Winds Meet*
- права на запись в папку игры  
  (в некоторых случаях может потребоваться запуск от имени администратора)

---

## Безопасность

- программа не собирает персональные данные
- не использует Steam API
- не изменяет исполняемые файлы игры
- работает только с файлами локализации

---

## Исходный код

Репозиторий содержит только исходный код лаунчера.  
Готовые `.exe` файлы публикуются в разделе **Releases**.
